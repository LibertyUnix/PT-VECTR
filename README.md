# VECTR
Automating Your Vectr Install with Vagrant & VMWare

# To get started
* git clone https://github.com/LibertyUnix/PT-VECTR
* cd VECTR
* vagrant up
* user - vagrant
* pw - vagrant
* To Start GUI - sudo systemctl isolate graphical
